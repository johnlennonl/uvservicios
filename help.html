Inicio rápido

Flujo recomendado de trabajo

1. Revisa el origen del archivo

Si el Excel del monitoreo diario viene mezclado, usa Preparador Excel antes de importarlo.

2. Carga o corrige en Gestión

Importa archivos limpios o crea entradas manuales cuando el dato no exista en el consolidado.

3. Confirma visualmente en Dashboard

Selecciona pozo, revisa gráficas, cambia fecha histórica y valida tendencias antes de compartir hallazgos.

4. Usa Data para auditoría

Consulta historial operativo o técnico, identifica vacíos y corrige registros puntuales.

5. Usa Estadísticas para seguimiento

Mide cobertura, pozos sin visita, carga diaria y rezagos por rango de fechas.

Módulo

Dashboard

Abrir Dashboard

Para qué sirve

  • Ver el comportamiento operativo de un pozo en tiempo real o sobre registros históricos.
  • Comparar contexto técnico, telemetría y variaciones antes de exportar un reporte.

Cómo usarlo

  1. Selecciona un pozo desde el filtro superior.
  2. Si quieres revisar otra medición, usa el selector histórico de fecha y hora.
  3. Pulsa refrescar cuando cambies pozo o quieras recargar datos.
  4. Activa el modo reporte cuando necesites exportar o imprimir resumen técnico.

Consejo operativo

Si un dato no aparece como esperas, primero valida en Data si el registro existe y luego confirma en Gestión si fue importado en la fecha correcta.

Módulo

Gestión

Abrir Gestión

Cuándo entrar aquí

  • Cuando vas a importar un archivo con monitoreos operativos.
  • Cuando necesitas cargar una medición técnica manual.
  • Cuando necesitas corregir o agregar un registro puntual.

Flujo recomendado

  1. Si el archivo está desordenado, pásalo primero por Preparador Excel.
  2. Importa el archivo o abre el formulario manual correspondiente.
  3. Confirma estado del pozo y fecha antes de guardar.
  4. Después valida el resultado en Dashboard o en Data.

Qué evita duplicados

La lógica de importación actual actualiza coincidencias exactas de pozo y fecha cuando aplica. Por eso es clave revisar que el archivo tenga fecha y pozo consistentes antes de cargar.

Módulo

Data

Abrir Data

Para qué sirve

  • Consultar la base histórica por pozo.
  • Ver si un pozo tiene registros, actividad reciente y tipo de historial.
  • Auditar si la información cargada coincide con lo esperado.

Pasos clave

  1. Busca el pozo en el selector principal.
  2. Usa el filtro de actividad reciente si quieres reducir la lista.
  3. Elige historial operativo o historial técnico según la revisión.
  4. Filtra por fecha para auditar un día puntual o revisar continuidad.

Uso recomendado

Data es la mejor vista para confirmar si el problema es de captura, de importación o de interpretación. Úsala como punto de auditoría antes de modificar datos.

Módulo

Estadísticas

Abrir Estadísticas

Para qué sirve

  • Medir cuántos monitoreos se hicieron por rango de fechas.
  • Identificar top de pozos visitados, cobertura del rango y alertas de rezago.
  • Explorar el detalle detrás de un KPI, un día o un pozo concreto.

Cómo leerlo

  1. Selecciona fecha inicio y fecha fin.
  2. Opcionalmente filtra uno o varios pozos.
  3. Genera el reporte y revisa KPIs, top pozos y alertas.
  4. Haz clic en tarjetas o barras para abrir el detalle de monitoreos.

Cuándo usarlo

Este módulo sirve mejor para seguimiento de supervisión y revisión administrativa que para corrección puntual de datos.

Módulo

Preparador de Excel

Abrir Preparador

Úsalo cuando

  • Recibes un consolidado con columnas sucias o formatos variables.
  • Necesitas exportar un archivo listo para Gestión sin tocar el original.
  • Quieres validar descartes antes de importar.

Secuencia de trabajo

  1. Carga el archivo XLSX, XLS o CSV.
  2. Revisa el resumen de filas válidas y descartadas.
  3. Valida la vista previa y descarga el archivo limpio.
  4. Ve a Gestión e importa ese archivo ya normalizado.

Resultado esperado

El archivo exportado debe conservar solo columnas útiles para el importador actual. Si el preview sale incoherente, corrige primero el origen en vez de forzar la carga.

Preguntas frecuentes

Solución rápida de dudas comunes

No veo un pozo en Dashboard o Data

Primero confirma en Data si el pozo existe en la lista. Si no aparece, revisa si fue cargado correctamente desde Gestión o si el archivo de origen lo traía con otro nombre.

Importé un archivo y no veo cambios

Revisa si la carga terminó sin errores, valida la fecha del registro y luego refresca la vista destino. Si el archivo fue limpiado antes, verifica la vista previa del Preparador.

¿Dónde corrijo un dato puntual?

Usa Gestión para crear o ajustar registros y luego confirma el resultado en Data o Dashboard según el caso.

¿Qué módulo uso primero?

Si vas a analizar, entra a Dashboard. Si vas a cargar datos, usa Gestión. Si el archivo está sucio, arranca en Preparador. Si quieres auditar históricos, usa Data. Si vas a medir cobertura, usa Estadísticas.
