Importar Producción Técnica

Sincroniza mediciones técnicas dentro del mismo flujo operativo, manteniendo el historial por pozo y fecha.

Cómo funciona esta sincronización

Si el Excel trae una medición nueva para un pozo en una fecha distinta, se agrega al historial técnico.

Si el Excel trae el mismo pozo con la misma fecha, esa medición se actualiza y no se duplica.

Las mediciones viejas se conservan. Solo cambia la coincidencia exacta de pozo + fecha.
