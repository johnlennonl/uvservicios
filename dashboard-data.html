Archivo CSV o Excel técnico

Carga masiva con columnas: CAMPO, POZO, EF, FECHA, POTENCIAL, BBPD, AYS, BNPD, CAT.

Importar Producción Técnica

Sincroniza mediciones técnicas dentro del mismo flujo operativo, manteniendo el historial por pozo y fecha.

Cómo funciona esta sincronización

Si el Excel trae una medición nueva para un pozo en una fecha distinta, se agrega al historial técnico.

Si el Excel trae el mismo pozo con la misma fecha, esa medición se actualiza y no se duplica.

Las mediciones viejas se conservan. Solo cambia la coincidencia exacta de pozo + fecha.

Gestión de Pozos

Configura la ficha fija BES por pozo para que aparezca en Dashboard, Data, Notificaciones y UVITO.

Cómo usar esta sección

Selecciona un pozo y registra manualmente su bomba, fabricante, motor, sensor y datos base. Esta ficha queda como referencia maestra del pozo.

Solo hace falta actualizarlo cuando cambie realmente el equipo instalado.
